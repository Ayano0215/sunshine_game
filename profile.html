前情提要

○ 為什麼想做這份遊戲？

想做這份遊戲的契機是為了復刻童年，除了傳承外，希望能對曾經喜歡過的遊戲加以創新，成為一個極具特色又不失傳統的遊戲。

 

 

劇情

四個班長為了爭奪班級榮譽，決定在此次的運動會上大展身手，為了班上拿下第一名。

 

 

遊戲介紹

 一、核心概念

1. 核心樂趣 :

(1) 多人競速對戰 : 與朋友或家人比拚手速與操作反應，擁有競賽樂趣。
(2) 單人挑戰紀錄 : 玩家可以挑戰自己或排行榜紀錄，附有成就感。

2. 主要特色 : 

(1) 懷舊復刻 : 致敬紅白機經典運動會遊戲，保留原始特色。
(2) 創新元素 : 加入地圖、新關卡或操作方式，使比賽變得比原本更豐富有趣。
(3) 公平競速 : 操作決定勝負，適合喜歡技術挑戰的玩家。
(4) 遊戲體驗目標 :　讓玩家體驗「懷舊樂趣+現代體驗手感」讓朋友或家人聚在一起也能玩得開心，或單獨挑戰自我紀錄。

二、遊戲玩法

1. 比賽項目

(1) 必做項目

  • 短跑：連打按鍵衝刺，考驗玩家手速與反應力。

  • 障礙賽：結合衝刺與時機判斷跳躍，增加操作多樣性。

  • 跳遠 / 跳高：透過速度蓄力與起跳判斷，挑戰玩家技巧。

(2) 額外挑戰

  • 游泳：節奏點擊控制速度。

  • 鉛球 / 標槍：蓄力與角度投擲，講求精準度。

(3) 趣味項目

  • 個人興趣：射鏢，輕鬆娛樂。

2. 遊戲模式設計

(1) 單人模式

  • 可挑戰自我紀錄並存檔。

  • 系統會記錄歷史最佳成績或排行榜。

  • 可進行連續多項目挑戰，結算總分數，提升成就感。

(2) 多人模式

  • 2~4人同場比賽，享受即時競速樂趣。

  • 系統即時計算每位玩家操作差異並結算排名，確保公平競技。

關卡設計

短跑

  • 核心玩法：玩家連打指定按鍵（或交替按兩鍵）衝刺到終點。

  • 道具需求：

    • 跑道場景（含起點 / 終點線）

    • 起跑信號燈或裁判音效

    • 玩家角色動畫（起跑、奔跑、衝線）

    • 倒數計時 UI

  • 操作模式：

    • 按鍵：交替連打 A + D（或左右方向鍵） → 速度越快角色跑越快

    • 起跑：倒數結束前提早按會「偷跑失誤」，延遲1秒起跑

    • 加速區間：維持穩定節奏觸發短暫加速特效

    • 體力/汗量槽：連打太快會過熱，需要節奏控制

    • 終點動畫：滑壘、撲倒、勝利姿勢等搞笑特寫

障礙賽

  • 核心玩法：跑步 + 時機跳躍，越過障礙避免摔倒。

  • 道具需求：

    • 跑道＋多個間隔不同欄架

    • 背景捲動或攝影機追蹤效果

    • 跳躍 / 絆倒動畫

  • 操作模式：

    • 移動：連打 A + D 前進

    • 跳躍：Space（空白鍵）在正確時機按下即可跳過障礙

    • 判定：太早或太晚撞欄、扣速度

    • 完美跳躍：短暫加速 + 視覺特效

    • 連續Perfect Combo

    • 搞笑動畫：飛過頭或摔倒滑行

跳遠 / 跳高

  • 核心玩法：節奏蓄力 + 把握起跳角度，跳得越遠越好。

  • 道具需求：

    • 跳遠沙坑 / 跳高橫桿

    • 蓄力條、角度指示器

    • 判定線動畫

  • 操作模式：

    • 蓄力：按住 Space → 蓄力條上升

    • 放開 Space：決定起跳力度

    • 角度調整：蓄力後出現角度條 → 約 45° 放開最佳

    • 落地判定：超線或起跳太早則失敗

    • 隨機事件：風速變化、場地濕滑

鉛球 / 標槍

  • 核心玩法：按時機蓄力、控制角度投出最遠距離。

  • 道具需求：

    • 投擲場地（含量測線）

    • 投擲物模型（鉛球 / 標槍）

    • 力量條、角度條

    • 投擲動畫

  • 操作模式：

    • 蓄力：按住 Space → 力量條上下波動

    • 放開：決定力度

    • 角度控制：方向鍵 ↑↓ 選角度（約 40~45°最佳）

    • 落地計算：拋物線軌跡 + 隨機風速影響

射鏢

  • 核心玩法：瞄準與穩定控制，考驗精準與手眼協調。

  • 道具需求：

    • 靶盤模型（含分數區）

    • 瞄準準星 / 搖晃特效

    • 鏢投擲音效與分數 UI

  • 操作模式：

    • 瞄準：滑鼠控制準星移動

    • 投擲：點擊左鍵，根據晃動節奏判定準度

    • 難度機制：瞄準圈隨時間震動加快
